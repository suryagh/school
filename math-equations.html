SAT Math Equations and Formulas

1. Algebra

  • Linear equations: y = mx + b (slope-intercept form), where m is the slope, and b is the y-intercept.
  • Slope formula: m = (y2 - y1) / (x2 - x1) (calculate the slope between two points).
  • Quadratic equation: ax² + bx + c = 0
  • Quadratic formula: x = (-b ± √(b² - 4ac)) / 2a
  • Factoring form: ax² + bx + c = (px + q)(rx + s)

2. Geometry

  • Area of a triangle: A = ½bh (base b, height h)
  • Area of a rectangle: A = lw (length l, width w)
  • Area of a circle: A = πr² (radius r)
  • Circumference of a circle: C = 2πr
  • Pythagorean theorem: a² + b² = c² (for right triangles)
  • Distance formula: d = √((x2 - x1)² + (y2 - y1)²)
  • Midpoint formula: Midpoint = ((x1 + x2) / 2, (y1 + y2) / 2)
  • Volume of a rectangular prism: V = lwh
  • Volume of a cylinder: V = πr²h
  • Volume of a sphere: V = 4/3πr³
  • Area of a trapezoid: A = ½(b1 + b2)h
  • Surface area of a cylinder: SA = 2πr² + 2πrh

3. Trigonometry

  • SOHCAHTOA (Trigonometric ratios):
    • sin(θ) = opposite / hypotenuse
    • cos(θ) = adjacent / hypotenuse
    • tan(θ) = opposite / adjacent
  • Sine and cosine identities: sin²(θ) + cos²(θ) = 1

4. Statistics and Probability

  • Mean (average): Mean = Sum of terms / Number of terms
  • Median: Middle value of an ordered set.
  • Mode: Most frequent value in a set.
  • Probability: P = Number of favorable outcomes / Total number of outcomes
  • Counting principle: Multiply the number of choices at each step.

5. Functions

  • Slope of a line: m = (y2 - y1) / (x2 - x1)
  • Linear function: f(x) = mx + b
  • Vertex form of a quadratic function: f(x) = a(x - h)² + k
  • Exponential growth/decay: y = a(1 ± r)ᵗ

6. Coordinate Geometry

  • Equation of a circle: (x - h)² + (y - k)² = r²
  • Equation of a parabola: y = ax² + bx + c
  • Distance between two points: d = √((x2 - x1)² + (y2 - y1)²)

7. Exponents and Radicals

  • Laws of exponents:
    • xᵃ · xᵇ = x⁽ᵃ + ᵇ⁾
    • (xᵃ)ᵇ = x⁽ᵃᵇ⁾
    • x⁰ = 1
    • x⁻ᵃ = 1/xᵃ
  • Square root: √a · √b = √(ab)

8. Logarithms

  • Logarithmic form: log_b(x) = y is equivalent to bʸ = x
  • Logarithmic properties:
    • log_b(xy) = log_b(x) + log_b(y)
    • log_b(x/y) = log_b(x) - log_b(y)
    • log_b(xⁿ) = n · log_b(x)

9. Proportions and Ratios

  • Direct proportion: x₁/y₁ = x₂/y₂
  • Inverse proportion: x₁ · y₁ = x₂ · y₂

10. Other Essential Equations

  • Simple interest: I = Prt
  • Compound interest: A = P(1 + r/n)ⁿᵗ
  • Average rate of change: (f(b) - f(a)) / (b - a)

11. Complex Numbers

  • Form of a complex number: z = a + bi, where i = √-1
  • Addition of complex numbers: (a + bi) + (c + di) = (a + c) + (b + d)i
  • Multiplication of complex numbers: (a + bi)(c + di) = (ac - bd) + (ad + bc)i
  • Conjugate of a complex number: Conjugate of (a + bi) = (a - bi)

12. Absolute Value

  • Definition: |x| = { x, if x ≥ 0 ; -x, if x < 0 }
  • Distance between two points: d = |x₂ - x₁| (on a number line).

13. Inequalities

  • Linear inequality: ax + b > c, ax + b ≥ c, ax + b < c, ax + b ≤ c
  • Absolute value inequality: |x - a| < b means a - b < x < a + b
  • System of inequalities: Set of inequalities solved together (often graphically).

14. Sequences

  • Arithmetic sequence: aₙ = a₁ + (n - 1)d
  • Geometric sequence: aₙ = a₁ · r⁽ⁿ - 1⁾
  • Sum of an arithmetic sequence: Sₙ = n/2 (a₁ + aₙ)
  • Sum of a geometric sequence: Sₙ = a₁(1 - rⁿ)/(1 - r)

15. Rational Equations

  • Solving rational equations: Clear denominators by multiplying both sides by the least common denominator (LCD) before solving.

16. Percentages

  • Percentage formula: Percentage = (Part / Whole) * 100
  • Percentage increase/decrease: Percentage change = ((New value - Old value) / Old value) * 100
  • Percent of a number: x% of y = (x / 100) * y

17. Exponent Rules

  • Product of powers: xᵃ · xᵇ = x⁽ᵃ + ᵇ⁾
  • Power of a power: (xᵃ)ᵇ = x⁽ᵃᵇ⁾
  • Power of a product: (xy)ᵃ = xᵃ yᵃ
  • Negative exponent: x⁻ᵃ = 1/xᵃ
  • Fractional exponents: x^(a/b) = b√(xᵃ)

18. Radicals

  • Simplifying radicals: √ab = √a · √b
  • Adding/subtracting radicals: a√b + c√b = (a + c)√b
  • Multiplying radicals: √a · √b = √ab
  • Rationalizing denominators: 1/√a = √a/a

19. Proportions and Similarity

  • Proportion formula: a/b = c/d (cross-multiply to solve).
  • Properties of similar triangles: Corresponding sides of similar triangles are proportional.
  • Area ratios of similar figures: Area ratio = (Side of one figure / Side of another figure)²

20. Motion and Rate

  • Distance formula: d = rt, where d is distance, r is rate or speed, and t is time.
  • Average speed: Average speed = Total distance / Total time

21. Work Problems

  • Work formula: 1/t₁ + 1/t₂ = 1/tₜₒₜₐₗ, where t₁ and t₂ are the individual times taken by two workers to complete a job, and tₜₒₜₐₗ is the time taken when working together.

22. Simple Interest

  • Interest formula: I = Prt, where I is interest, P is the principal, r is the rate, and t is the time in years.

23. Compound Interest

  • Compound interest formula: A = P(1 + r/n)ⁿᵗ, where A is the amount, P is the principal, r is the annual interest rate, n is the number of times interest is compounded per year, and t is the time in years.

24. Circle Geometry

  • Arc length: L = (θ / 360) * 2πr, where θ is the angle in degrees.
  • Area of a sector: A = (θ / 360) * πr²
  • Equation of a circle: (x - h)² + (y - k)² = r²

25. Probability and Statistics

  • Probability formula: P(Event) = Number of favorable outcomes / Total number of outcomes
  • Expected value: E(X) = Σ [x · P(x)], where x is the value and P(x) is the probability of that value.
  • Permutation formula: P(n, r) = n! / (n - r)!
  • Combination formula: C(n, r) = n! / (r!(n - r)!)

26. Radians and Degrees

  • Conversion between degrees and radians:
    • 1 radian = 180° / π
    • 1° = π / 180 radians
  • Arc length in radians: L = rθ (where θ is in radians).
  • Area of a sector in radians: A = ½r²θ

27. Exponential Growth and Decay

  • Exponential growth: y = a(1 + r)ᵗ
  • Exponential decay: y = a(1 - r)ᵗ
  • General exponential function: y = abˣ

28. Parabolas and Conics

  • Standard form of a parabola: y = ax² + bx + c
  • Vertex form of a parabola: y = a(x - h)² + k
  • Focus and directrix of a parabola: The distance between the vertex and the focus or directrix is 1 / 4a.

29. Ellipses and Hyperbolas

  • Equation of an ellipse: (x - h)² / a² + (y - k)² / b² = 1
  • Equation of a hyperbola: (x - h)² / a² - (y - k)² / b² = 1

30. Piecewise Functions

  • Definition: A piecewise function is defined by multiple sub-functions, each applied to a different part of the domain.
  • Example:
    • f(x) = { 2x + 3, if x < 0 ; x² - 1, if x ≥ 0 }

31. Inequalities with Absolute Value

  • Absolute value inequalities: To solve |x| < a, rewrite as -a < x < a.
  • For |x| > a, rewrite as x < -a or x > a.

32. Logarithmic Functions

  • Logarithmic equation: log_b(x) = y implies bʸ = x
  • Change of base formula: log_b(x) = log_a(x) / log_a(b)
  • Logarithmic properties:
    • log_b(xy) = log_b(x) + log_b(y)
    • log_b(x / y) = log_b(x) - log_b(y)
    • log_b(xⁿ) = n · log_b(x)

33. Transformations of Functions

  • Vertical shift: f(x) + c shifts upward by c units.
  • Horizontal shift: f(x + c) shifts left by c units.
  • Reflection: -f(x) reflects over the x-axis; f(-x) reflects over the y-axis.
  • Vertical stretch/compression: a f(x), where a > 1 stretches, and 0 < a < 1 compresses.

34. Inverse Functions

  • Definition: If f(x) and g(x) are inverses, then f(g(x)) = g(f(x)) = x.
  • Finding the inverse: Swap x and y in the equation of the function and solve for y.

35. Matrix Operations

  • Matrix addition: Add corresponding elements.
  • Matrix multiplication: Multiply rows by columns and sum the products.
  • Determinant of a 2x2 matrix:
    • det [ [a, b], [c, d] ] = ad - bc

36. Vectors

  • Vector addition: v + w = (v₁ + w₁, v₂ + w₂)
  • Magnitude of a vector: |v| = √(v₁² + v₂²)
  • Dot product: v · w = v₁w₁ + v₂w₂

37. Permutation and Combination

  • Permutation: P(n, r) = n! / (n - r)!
  • Combination: C(n, r) = n! / (r!(n - r)!)

38. Counting Principles

  • Fundamental Counting Principle: If one event can occur in m ways and another can occur in n ways, the total number of ways the two events can occur is m · n.
  • Factorial: n! = n · (n - 1) · (n - 2) · ... · 1

39. Law of Sines and Cosines

  • Law of Sines: sin(A) / a = sin(B) / b = sin(C) / c
  • Law of Cosines: c² = a² + b² - 2ab·cos(C)

40. Unit Circle and Trig Identities

  • Unit circle values: Memorize the sine and cosine values of angles like 0°, 30°, 45°, 60°, 90°, etc.
  • Pythagorean identity: sin²(θ) + cos²(θ) = 1
  • Double angle identities:
    • sin(2θ) = 2sin(θ)cos(θ)
    • cos(2θ) = cos²(θ) - sin²(θ)

41. Parametric Equations

  • Definition: Parametric equations express coordinates as functions of a parameter, t:
    • x = f(t), y = g(t)
  • Converting to rectangular form: Eliminate the parameter t to get an equation involving only x and y.

42. Polar Coordinates

  • Converting polar to rectangular coordinates:
    • x = r·cos(θ)
    • y = r·sin(θ)
  • Converting rectangular to polar:
    • r = √(x² + y²)
    • θ = tan⁻¹(y / x)
